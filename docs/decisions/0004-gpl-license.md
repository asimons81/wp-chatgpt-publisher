# ADR 0004: Single GPL-2.0-or-later license

Status: accepted — 2026-07-12.

License the complete monorepo GPL-2.0-or-later. This avoids split-distribution ambiguity, aligns with WordPress.org's GPL-compatibility requirement, and gives contributors one clear inbound/outbound license. Release SBOM review remains mandatory because dependencies and assets retain their own compatible licenses.
