# Roadmap

## v1.1 candidates

- Stable AIOSEO public write adapter if upstream support is confirmed
- Shared distributed rate-limit store and administrative metrics export
- Richer block round-trip fidelity and structured block editing
- WordPress 7.x client-side Abilities API integration where it improves admin UX
- Optional `search`/`fetch` aliases for ChatGPT company-knowledge compatibility without changing the product's WordPress-first tool names
- Multisite design investigation, not implementation until permission boundaries are proven
- Additional localization and accessibility audits

Non-goals remain WooCommerce mutation, broad site administration, code/SQL/filesystem execution, permanent deletion, autonomous generation/publishing, built-in LLM calls, and proprietary hosting requirements.
