# 75-second reviewer demo script

Use only the disposable reviewer site and synthetic content. Keep credentials, OAuth codes, tokens, and admin-only URLs out of frame.

| Time      | Visual                                   | Narration / action                                                                                                                                            |
| --------- | ---------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 0:00-0:08 | ChatGPT app enabled; connected-site card | "Editorial Publisher connects ChatGPT to a WordPress user's existing authority. It needs no OpenAI API key and never receives the user's WordPress password." |
| 0:08-0:18 | Search results, then selected content    | Ask: "Find the release safety article and summarize its metadata." Show compact search first, then selected retrieval.                                        |
| 0:18-0:32 | Draft result and review card             | Ask to create a clearly named disposable draft from supplied Markdown. Show status Draft, word count, taxonomy, featured image, and SEO review.               |
| 0:32-0:43 | Media/SEO/taxonomy updates               | Assign the existing Release Safety category and MCP tag, attach the supplied benign image, and set its alt text plus native SEO description.                  |
| 0:43-0:52 | Editorial publish denial                 | Attempt to publish while using the Editorial connection. Highlight the specific scope-denied remediation.                                                     |
| 0:52-1:05 | Publisher confirmation                   | Switch to the pre-approved Publisher connection, request publish, review warnings and intended action, then click **Confirm publish** once.                   |
| 1:05-1:12 | Result and replay denial                 | Open the public result, then demonstrate that replaying the confirmation is rejected.                                                                         |
| 1:12-1:15 | WordPress connections page               | Revoke the disposable connection and state that future calls now fail independently at WordPress.                                                             |

Capture one desktop and one mobile pass. Add captions, avoid cuts that imply autonomous publishing, and end on the open-source repository/license rather than an endorsement claim.
